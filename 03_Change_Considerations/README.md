# Change Considerations

This folder contains two documents covering the intervention 
analysis and recommended solution for the onboarding process.

## Contents

### 1. BP Change Considerations
Documents all possible interventions identified in response 
to the process problems and risks, categorised as:

- **Controls** are the system-enforced rules that prevent failures
- **Process Improvements** are changes to how work is done
- **Process Redesigns** refers to the structural redesign of the workflow
- **Automation Opportunities** refers to all the tasks suitable for automation

The recommended solution is identified as a unified 
ServiceNow-based customer onboarding portal supported by 
standard operating procedures and work instructions.

### 2. Mapping of Interventions to Problems and Risks
A three-column mapping table showing:
- The problem or risk identified
- The recommended intervention
- How the intervention resolves the issue

## Why This Matters
This section demonstrates that the recommended solution was 
not chosen arbitrarily.It was selected because it addresses 
the highest-priority problems and risks identified through 
structured stakeholder analysis.
