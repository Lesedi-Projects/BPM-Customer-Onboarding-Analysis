# Change Considerations

This folder contains two documents:

1. **Business Process Change Considerations**
 All possible interventions considered, categorised as controls, process improvements, process redesigns, and automation opportunities. This includes the recommended intervention which is a consolidated ServiceNow-based onboarding portal supported by SOPs and work instructions.

2. **Mapping of Interventions to Problems and Risks**
 A three-column mapping table showing each recommended intervention, the problem or risk it addresses, and how it resolves the issue.
