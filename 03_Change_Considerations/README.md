# Change Considerations

This folder contains two documents:

1. **BP_Change_Considerations** — All possible interventions considered, categorised as controls, process improvements, process redesigns, and automation opportunities. Includes the recommended intervention: a unified ServiceNow-based onboarding portal supported by SOPs and work instructions.

2. **Mapping_of_Interventions_to_Problems_and_Risks** — A three-column mapping table showing each recommended intervention, the problem or risk it addresses, and how it resolves the issue.
