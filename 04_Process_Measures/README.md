# Process Measures and KPIs

This folder contains the proposed KPI framework for measuring 
the performance of the redesigned onboarding process.

## What is a KPI?
A KPI (Key Performance Indicator) is a measurable value that 
shows whether a process is achieving its objectives. KPIs 
allow organisations to track improvement over time and identify 
when performance is falling short of expectations.

## KPIs Included

| KPI | What It Measures |
|---|---|
| First-submission completeness rate | How many applications are submitted correctly first time |
| Onboarding cycle time | Time from complete application receipt to account activation |
| SOP compliance rate | Whether staff follow the new standard procedures |
| Portal adoption rate | Whether submissions move from email to the portal |
| Rework rate | How often applications are returned for correction |
| Status visibility rate | Whether customers can track their application progress |

## How Each KPI is Presented
Each KPI includes:
- **As-Is baseline** : Current performance drawn from 
  stakeholder interview evidence
- **To-Be target**: Desired performance after implementation
- **Rationale** : Why the target is set at that level

## Note
Specific baseline figures reflecting internal organisational
