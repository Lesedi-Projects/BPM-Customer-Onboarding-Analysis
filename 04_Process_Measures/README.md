# Process Measures and KPIs

This folder contains the proposed KPI framework for the redesigned onboarding process.

Each KPI includes:
- As-is baseline drawn from qualitative interview evidence
- To-be target following implementation
- Rationale for the target

The KPIs cover the following : first-submission completeness rate, onboarding cycle time, SOP compliance rate, portal adoption rate, rework rate, and status visibility rate.
